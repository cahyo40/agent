# Instructional Designer

## Overview

Design effective learning experiences using proven pedagogical methods. Covers curriculum design, lesson planning, module creation, and tutorial writing.

## When to Use This Skill

- Use when creating curriculum
- Use when designing lesson plans
- Use when setting learning objectives
- Use when creating learning modules
- Use when writing tutorials

## How It Works

### Step 1: ADDIE Model

```markdown
## Instructional Design Framework

### A - Analysis
- Who are the learners?
- What are their current skills?
- What do they need to learn?
- What constraints exist?

### D - Design
- Define learning objectives
- Sequence content
- Choose instructional strategies
- Plan assessments

### D - Development
- Create content materials
- Develop activities
- Build assessments
- Create instructor guides

### I - Implementation
- Pilot test
- Train instructors
- Deliver course
- Provide support

### E - Evaluation
- Collect feedback
- Measure outcomes
- Analyze results
- Revise and improve
```

### Step 2: Learning Objectives (SMART)

```markdown
## Writing Learning Objectives

### Bloom's Taxonomy Verbs

| Level | Verbs | Example |
|-------|-------|---------|
| Remember | List, define, recall | "List 5 principles of..." |
| Understand | Explain, describe, summarize | "Explain how X works" |
| Apply | Use, implement, solve | "Apply the formula to..." |
| Analyze | Compare, differentiate, examine | "Analyze the differences..." |
| Evaluate | Judge, critique, assess | "Evaluate the effectiveness..." |
| Create | Design, develop, produce | "Create a solution for..." |

### Objective Template
"By the end of this [lesson/module], learners will be able to 
[action verb] + [specific content] + [condition/context]"

### Example
"By the end of this module, learners will be able to 
CREATE a responsive website using HTML and CSS 
without referring to documentation."
```

### Step 7: Engagement Strategies

```markdown
## Active Learning Techniques

### Think-Pair-Share
1. Think individually (2 min)
2. Pair with neighbor (3 min)
3. Share with class (5 min)

### Chunking Content
- 10-15 min content blocks
- Activity between chunks
- Recap after each chunk

### Scaffolding
- Start with guided practice
- Gradually reduce support
- End with independent practice
```
