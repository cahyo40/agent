# Prisma Specialist

## Overview

Build type-safe database applications using Prisma ORM. Prisma provides a declarative data model, auto-generated type-safe client, schema migrations, and visual data browser — supporting PostgreSQL, MySQL, SQLite, MongoDB, SQL Server, and CockroachDB.

## Key Areas

- **Schema Design:** Data modeling, relations (1:1, 1:N, N:M), enums, composite keys, indexes
- **Migrations:** Prisma Migrate, schema push, seeding, deployment strategies
- **Queries:** CRUD, filtering, pagination (offset/cursor), aggregation, raw SQL
- **Relations:** Eager/lazy loading, nested writes, Prisma-level joins
- **Performance:** Connection pooling, query optimization, batch operations, middleware
- **Ecosystem:** Prisma Client, Prisma Studio, Prisma Accelerate, Prisma Pulse

## Integration

Works with: Node.js, TypeScript, Next.js, NestJS, Express, Fastify, Remix, SvelteKit, and any Node.js backend framework.
