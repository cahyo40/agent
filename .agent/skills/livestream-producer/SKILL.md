# Livestream Producer

## Overview

Produce professional livestreams with engaging audience interaction.

## When to Use This Skill

- Use when setting up streams
- Use when going live

## How It Works

### Step 1: OBS Setup

```markdown
## Stream Settings

### Video
- Resolution: 1920x1080
- FPS: 30 (stable) or 60 (gaming)
- Bitrate: 4500-6000 Kbps

### Audio
- Sample rate: 48 kHz
- Bitrate: 160 Kbps
- Desktop + Mic sources

### Scenes
- Starting Soon
- Main Content
- BRB (Be Right Back)
- Ending
```

### Step 2: Scene Layout

```markdown
## Common Layouts

### Talking Head
- Camera: 70%
- Chat: 20%
- Alerts: 10%

### Gaming
- Game: 80%
- Camera: 15% (corner)
- Chat overlay: 5%

### Interview
- Split screen 50/50
- Lower thirds for names
```

### Step 3: Engagement

```markdown
## Audience Interaction

### Chat Management
- Acknowledge by name
- Answer questions live
- Use polls/votes
- Highlight super chats

### Retention
- Tease upcoming content
- Give reasons to stay
- Regular interaction (every 5 min)
```

## Best Practices

- ✅ Test before going live
- ✅ Stable internet (wired)
- ✅ Backup stream key
- ❌ Don't ignore chat
- ❌ Don't stream without plan
