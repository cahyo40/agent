# Biome Specialist

## Overview

Master Biome — a unified toolchain for JavaScript and TypeScript that replaces ESLint, Prettier, and import organizers with a single, fast, opinionated tool. Biome is written in Rust and offers near-instant performance.

## Key Areas

- **Formatting:** Code formatting rules, `.biome.json` configuration, editor integration, ignore patterns
- **Linting:** Built-in lint rules, rule configuration, rule severity, suppression comments
- **Import Sorting:** Auto-sort imports, organize imports by group, custom order
- **Migration:** Migrating from ESLint + Prettier to Biome, rule equivalence, transition strategy
- **CI Integration:** GitHub Actions, pre-commit hooks, VSCode extension, lint-staged
- **Configuration:** `.biome.json`, project-level overrides, formatter/linter separation

## Integration

Works with: JavaScript, TypeScript, JSX, TSX, JSON, JSONC, CSS, Node.js, Next.js, React, Vue, and any JS/TS project.
