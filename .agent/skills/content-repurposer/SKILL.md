# Content Repurposer

## Overview

This skill transforms you into a **Content Multiplication Specialist**. You will master **Format Transformation**, **Platform Adaptation**, and **Content Atomization** for maximizing the reach of every piece of content you create.

## When to Use This Skill

- Use when maximizing content ROI
- Use when adapting long-form to short-form
- Use when posting across multiple platforms
- Use when creating content calendars
- Use when building content flywheels

---

## Part 1: The Repurposing Framework

### 1.1 Content Atomization

One piece of "pillar" content becomes many "atoms".

```
YouTube Video (20 min)
    ├── Blog Post (2000 words)
    ├── 5 TikTok clips (30-60 sec each)
    ├── 3 Instagram Reels
    ├── 10 Twitter/X posts
    ├── LinkedIn article
    ├── Email newsletter
    ├── Podcast episode (audio extract)
    └── Quote graphics (5 images)
```

### 1.2 The Pillar Content Principle

Create ONE comprehensive piece, then derive everything else.

| Pillar Format | Best For |
|---------------|----------|
| **Long Video** | Interviews, tutorials |
| **Long Blog** | SEO, evergreen content |
| **Podcast Episode** | Conversations, deep dives |
| **Webinar** | Live teaching, Q&A |

## Part 2: Format Transformations

### 2.1 Video → Multiple Formats

| Output | How |
|--------|-----|
| **Short Clips** | Extract best 30-60 sec moments |
| **Blog Post** | Transcribe and edit transcript |
| **Audiogram** | Waveform video with audio |
| **Quote Graphics** | Key quotes as images |
| **Thread** | Key points as Twitter thread |
| **Carousel** | Summary slides for Instagram |

### 2.2 Blog → Multiple Formats

| Output | How |
|--------|-----|
| **Video Script** | Adapt for talking head/voiceover |
| **Thread** | Break into 5-12 key points |
| **Infographic** | Visualize data/steps |
| **Email** | Summary + link to full post |
| **Slides** | LinkedIn carousel |
| **Audio** | Record as podcast/note |

### 2.3 Podcast → Multiple Formats

| Output | How |
|--------|-----|
| **Video Podcast** | Add static image or video recording |
| **Clips** | Best quotes with captions |
| **Show Notes** | Summarized blog post |
| **Newsletter** | Episode highlights + link |

## Part 3: Platform Adaptation

### 3.1 Same Message, Different Format

| Platform | Format | Tone |
|----------|--------|------|
| **TikTok/Reels** | Vertical video, 30-60s | Casual, hook-driven |
| **YouTube** | Horizontal, 8-20 min | Educational, polished |
| **Twitter/X** | Text threads, 5-15 tweets | Concise, punchy |
| **LinkedIn** | Text posts, carousels | Professional, insightful |
| **Instagram** | Carousels, Stories | Visual, aesthetic |
| **Email** | Long-form text | Personal, direct |

### 3.2 Native Feel

Don't just cross-post. Adapt:

- Rewrite copy for each platform's voice.
- Use platform-native features (polls, stickers).
- Adjust aspect ratio (16:9, 9:16, 1:1).

## Part 4: Workflow & Tools

### 4.1 Repurposing Workflow

1. **Create Pillar Content**: Video, long blog, podcast.
2. **Transcribe**: Otter.ai, Descript.
3. **Extract Moments**: Find 5-10 clip-worthy sections.
4. **Adapt Copy**: Rewrite for each platform.
5. **Create Visuals**: Graphics, carousels.
6. **Schedule**: Buffer, Hootsuite, Later.
7. **Analyze**: What performed best? Double down.

### 4.2 Tools

| Tool | Purpose |
|------|---------|
| **Descript** | Transcription, video editing |
| **Opus Clip** | AI clip extraction |
| **Canva** | Graphics, carousels |
| **Buffer/Hootsuite** | Scheduling |
| **Repurpose.io** | Automated cross-posting |
| **Notion** | Content calendar |

## Part 5: Content Calendar

### 5.1 Weekly Cadence Example

| Day | Platform | Content |
|-----|----------|---------|
| **Monday** | YouTube | Pillar video drops |
| **Tuesday** | Blog | Written version published |
| **Wednesday** | Twitter | Thread from blog |
| **Thursday** | Instagram | Carousel from key points |
| **Friday** | TikTok | 3 clips from video |
| **Weekend** | Email | Newsletter summary |

### 5.2 Evergreen vs Timely

- **Evergreen**: Repurpose forever (tutorials, how-tos).
- **Timely**: News, trends (shorter shelf life).

## Part 6: Best Practices Checklist

### ✅ Do This

- ✅ **Start with Quality Pillar**: Garbage in = garbage out.
- ✅ **Batch Create**: Repurpose all at once.
- ✅ **Track Performance**: Know which atoms drive traffic.

### ❌ Avoid This

- ❌ **Identical Cross-Posting**: Feels lazy, algorithms punish.
- ❌ **Ignoring Platform Norms**: Each platform has culture.
- ❌ **Repurposing Too Early**: Wait for pillar to perform first.
