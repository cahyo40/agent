# Social Media Marketer

## Overview

This skill helps you develop social media strategies, create engaging content, and run effective paid ad campaigns.

## When to Use This Skill

- Use when creating social content
- Use when planning ad campaigns
- Use when developing content calendars

## How It Works

### Step 1: Content Strategy

```markdown
## Social Media Content Strategy

### Platform Mix
| Platform | Audience | Content Type | Frequency |
|----------|----------|--------------|-----------|
| Twitter/X | Tech/Developers | Tips, threads, news | 3-5x/day |
| LinkedIn | B2B/Professionals | Articles, insights | 1x/day |
| Instagram | Younger audience | Visual, stories | 1-2x/day |
| TikTok | Gen Z/Millennials | Short video | 1-3x/day |
| YouTube | All ages | Long-form, tutorials | 2x/week |

### Content Pillars (4-5 themes)
1. **Educational** (40%) - Tips, tutorials, how-tos
2. **Engagement** (25%) - Questions, polls, memes
3. **Promotional** (20%) - Product features, launches
4. **Behind-the-scenes** (10%) - Team, culture, process
5. **User-generated** (5%) - Testimonials, reposts

### Content Mix Formula
- 80% Value content (educate, entertain, inspire)
- 20% Promotional content (product, offers, CTAs)
```
