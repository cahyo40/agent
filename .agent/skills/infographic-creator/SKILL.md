# Infographic Creator

## Overview

Create shareable infographics and data visualizations.

## When to Use This Skill

- Use when visualizing data
- Use when creating shareable graphics

## How It Works

### Step 1: Infographic Types

```markdown
## Common Formats

### Statistical
- Key numbers/stats
- Comparisons
- Trends over time

### Process/Timeline
- Step-by-step flows
- Historical events
- Project milestones

### Comparison
- Pros vs Cons
- Before vs After
- Product comparisons

### List/Checklist
- Tips and tricks
- Resource lists
- Quick guides
```

### Step 2: Design Principles

```markdown
## Visual Hierarchy

1. Title (largest, top)
2. Key stat/hero visual
3. Supporting data
4. Source/branding

## Layout Grid
- Header: 15%
- Body: 70%
- Footer: 15%

## Color Rules
- 1 primary color
- 2-3 accent colors
- Neutral for text
```

### Step 3: Chart Selection

```markdown
| Data Type | Best Chart |
|-----------|------------|
| Comparison | Bar chart |
| Trend | Line chart |
| Part of whole | Pie/Donut |
| Distribution | Histogram |
| Relationship | Scatter |
```

## Best Practices

- ✅ One main message
- ✅ Readable at 50% size
- ✅ Include data sources
- ❌ Don't clutter
- ❌ Don't use 3D charts
