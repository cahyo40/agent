## When to Use This Skill

- Use when planning YouTube content strategy
- Use when writing video scripts

- Use when designing thumbnail concepts
- Use when analyzing channel performance
- Use when the user asks about YouTube growth

## Common Pitfalls

**Problem:** Low click-through rate (CTR)
**Solution:** Improve thumbnail contrast, faces, and emotional titles. A/B test variations.

**Problem:** Poor audience retention
**Solution:** Stronger hook, more pattern interrupts, cut dead air, use visuals.

**Problem:** Not getting recommended
**Solution:** Increase session timeâ€”create playlists, end screens to related videos.
