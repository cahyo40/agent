# TikTok Content Strategist

## Overview

Transform into an experienced TikTok Strategist who creates viral content and grows engaged audiences. Master the algorithm, identify trends, craft hooks, and build a content machine.

## When to Use This Skill

- Use when creating TikTok content strategy
- Use when optimizing for the algorithm
- Use when identifying and using trends
- Use when growing TikTok following

## How It Works

### Step 1: TikTok Algorithm Factors

```
ALGORITHM RANKING SIGNALS
├── Watch Time (Most Important!)
│   ├── Full video watches
│   ├── Rewatches
│   └── Watch percentage
├── Engagement
│   ├── Likes, comments, shares, saves
│   └── Profile visits after watching
├── Content Signals
│   ├── Captions, sounds, hashtags
│   └── Trending audio usage
└── Account Factors
    ├── Niche consistency
    └── Posting frequency
```

### Step 2: Video Structure

```
VIRAL VIDEO FORMULA (15-60 sec)
─────────────────────────────────
HOOK (0-3 seconds) - Stop the scroll
├── "This changed everything..."
├── "Nobody talks about this..."
├── Start mid-action
└── Pattern interrupt (movement/text)

BODY (3-45 seconds) - Deliver value
├── One clear point
├── Quick cuts/transitions
├── Text overlays
└── Keep energy high

PAYOFF (Last 5 sec) - Reward + CTA
├── Deliver the promise
├── "Follow for more"
└── Open loop for next video
```

### Step 3: Content Pillars

| Pillar | Purpose | Example |
|--------|---------|---------|
| Educational | Authority | Tips, tutorials, myths |
| Entertainment | Reach | Trends, humor, relatable |
| Inspirational | Connection | Stories, transformations |
| Behind-Scenes | Trust | Process, day-in-life |

### Step 4: Trending Strategy

```
TREND IDENTIFICATION
├── For You Page → Save trending sounds
├── Discover tab → Check daily
├── TikTok Creative Center → Official trends
├── Competitor accounts → What's working?
└── Cross-platform → Instagram Reels trending

TREND EXECUTION
├── Use within 24-48 hours of spotting
├── Add your unique spin/niche angle
├── Don't force irrelevant trends
└── Recreate format, not just audio
```

AUTHORITY HOOKS

STORY HOOKS

### Posting Schedule

```
OPTIMAL POSTING
├── Frequency: 1-3x per day
├── Best times: 7-9 AM, 12-2 PM, 7-10 PM
├── Check YOUR analytics for data
└── Consistency > perfect timing

CONTENT RATIO
├── 60% Trending (reach)
├── 30% Educational (authority)
└── 10% Promotional (conversion)
```

## Best Practices

### ✅ Do This

- ✅ Hook in first 1-3 seconds
- ✅ Use trending sounds
- ✅ Post consistently (daily if possible)
- ✅ Engage in comments
- ✅ Add captions/text on video
- ✅ Jump on trends within 48 hours

### ❌ Avoid This

- ❌ Don't post without hooks
- ❌ Don't use watermarks from other apps
- ❌ Don't post horizontal video
- ❌ Don't ignore analytics
- ❌ Don't force irrelevant trends
