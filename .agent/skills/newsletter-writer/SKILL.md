# Newsletter Writer

## Overview

This skill transforms you into an **Email Content Specialist**. You will master **Subject Lines**, **Email Structure**, **Subscriber Growth**, and **Retention Strategies** for building a successful newsletter.

## When to Use This Skill

- Use when writing email newsletters
- Use when growing subscriber lists
- Use when optimizing open and click rates
- Use when monetizing newsletters
- Use when building personal brand through email

---

## Part 1: Newsletter Fundamentals

### 1.1 Why Newsletters?

| Benefit | Explanation |
|---------|-------------|
| **Ownership** | You own the email list (unlike social) |
| **Direct Access** | Lands in inbox, no algorithm |
| **Monetizable** | Ads, sponsorships, paid tiers |
| **Trust Building** | Regular touchpoint builds relationship |

### 1.2 Newsletter Types

| Type | Cadence | Example |
|------|---------|---------|
| **Curated** | Weekly | Links + commentary |
| **Original Essays** | Weekly | Long-form thoughts |
| **News Digest** | Daily | Industry updates |
| **Tutorial** | Weekly | How-to content |
| **Community** | Bi-weekly | Member spotlights |

## Part 2: Subject Lines

### 2.1 The 5-Word Rule

Subject lines should be scannable in 5 words.

### 2.2 Subject Line Formulas

| Formula | Example |
|---------|---------|
| **Curiosity** | "The weird trick that actually works" |
| **Number** | "5 tools I use daily" |
| **Question** | "Are you making this mistake?" |
| **How-To** | "How to write better emails" |
| **Personal** | "I need your help with something" |
| **Urgency** | "Last chance: [Offer] ends tonight" |

### 2.3 A/B Testing

Test subject lines with 20% of list, send winner to 80%.

## Part 3: Email Structure

### 3.1 Anatomy of a Newsletter

| Section | Purpose |
|---------|---------|
| **Subject** | Get the open |
| **Preview Text** | Extend subject line |
| **Hook** | First 2 sentences (above the fold) |
| **Body** | Value delivery |
| **CTA** | What to do next |
| **P.S.** | Secondary CTA or personal note |

### 3.2 Formatting Tips

- **Short Paragraphs**: 1-3 sentences.
- **Subheadings**: Break up sections.
- **Bold Key Points**: Scanners skim.
- **Links**: 3-5 maximum per email.
- **Images**: Use sparingly (can hurt deliverability).

## Part 5: Metrics & Optimization

### 5.1 Key Metrics

| Metric | Benchmark |
|--------|-----------|
| **Open Rate** | 40-50% (engaged list) |
| **Click Rate** | 2-5% |
| **Unsubscribe Rate** | < 0.5% per email |
| **Reply Rate** | 1-2% (good engagement) |

### 5.2 Improving Metrics

- **Low Opens**: Better subject lines, send time testing.
- **Low Clicks**: Clearer CTAs, more relevant links.
- **High Unsubs**: Content mismatch, too frequent.

### 6.2 Pricing Sponsors

Formula: `CPM × List Size / 1000`

- CPM for newsletters: $30-$100+

## Part 7: Best Practices Checklist

### ✅ Do This

- ✅ **Be Consistent**: Same day, same time weekly.
- ✅ **Write Like a Friend**: Conversational tone wins.
- ✅ **Clean Your List**: Remove inactive subscribers.

### ❌ Avoid This

- ❌ **Buying Email Lists**: Destroys deliverability.
- ❌ **Too Many CTAs**: One primary CTA per email.
- ❌ **Ignoring Mobile**: 60%+ read on phone.
