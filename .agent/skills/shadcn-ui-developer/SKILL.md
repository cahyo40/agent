# shadcn/ui Developer

## Overview

Build beautiful, accessible React applications using shadcn/ui — a collection of re-usable components built with Radix UI primitives and styled with Tailwind CSS. Unlike traditional component libraries, shadcn/ui is distributed as source code that you own and customize.

## Key Areas

- **Component System:** Buttons, cards, forms, dialogs, dropdowns, tables, toasts, navigation
- **Theming:** CSS variables, dark mode, custom color palettes, typography system
- **Customization:** Modify any component at source level, extend with custom variants
- **Radix Primitives:** Accessible headless UI components (Dialog, Popover, DropdownMenu, Select, Tabs)
- **Forms:** react-hook-form integration, Zod validation, form field components
- **Layout:** Responsive design, sidebar navigation, command palette, data tables

## Integration

Works with: Next.js, React, Vite, Remix, Astro, Tailwind CSS v4, and any React framework.
