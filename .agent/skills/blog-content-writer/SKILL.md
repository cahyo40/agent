# Blog Content Writer

## Overview

This skill transforms you into an **SEO Content Strategist**. You will master **Content Structure**, **Keyword Optimization**, **Engagement Writing**, and **Readability** for creating blog posts that rank and convert.

## When to Use This Skill

- Use when writing long-form blog articles
- Use when optimizing content for search engines
- Use when structuring how-to guides and tutorials
- Use when improving content readability
- Use when building topical authority

---

## Part 1: Content Structure

### 1.1 Anatomy of a Blog Post

| Section | Purpose | Length |
|---------|---------|--------|
| **Hook** | Grab attention | 2-3 sentences |
| **Introduction** | Set expectations, promise value | 100-150 words |
| **Body** | Deliver on promise (H2/H3 sections) | 800-2000 words |
| **Conclusion** | Summarize, call to action | 100-150 words |

### 1.2 The AIDA Framework

1. **Attention**: Hook with a bold statement or question.
2. **Interest**: Present the problem/opportunity.
3. **Desire**: Show the solution and benefits.
4. **Action**: Tell them what to do next.

### 1.3 Heading Structure

```
H1: Main Title (only one)
  H2: Major section
    H3: Subsection
    H3: Subsection
  H2: Major section
    H3: Subsection
```

## Part 2: SEO Optimization

### 2.1 On-Page SEO Checklist

| Element | Best Practice |
|---------|---------------|
| **Title Tag** | Keyword near front, < 60 chars |
| **Meta Description** | Compelling, < 160 chars |
| **URL Slug** | Short, keyword-rich |
| **H1** | Contains primary keyword |
| **H2s** | Include secondary keywords |
| **Image Alt Text** | Descriptive, keyword if natural |
| **Internal Links** | 3-5 links to related posts |
| **External Links** | 1-3 to authoritative sources |

### 2.2 Keyword Placement

- **Title**: Must include primary keyword.
- **First 100 words**: Natural inclusion.
- **Subheadings**: Where relevant.
- **Throughout**: 1-2% density (don't stuff).

### 2.3 Featured Snippet Optimization

| Snippet Type | Format |
|--------------|--------|
| **Definition** | "X is..." paragraph under H2 |
| **List** | Numbered/bulleted list under H2 |
| **Table** | Comparison table |
| **Steps** | "How to" with numbered steps |

## Part 3: Writing Techniques

### 3.1 Readability

- **Short Sentences**: < 20 words average.
- **Short Paragraphs**: 2-4 sentences.
- **Active Voice**: "Do X" not "X should be done".
- **Grade Level**: Aim for 8th grade (Hemingway App).

### 3.2 Engagement Techniques

| Technique | Example |
|-----------|---------|
| **Questions** | "Ever wondered why...?" |
| **You/Your** | Address reader directly |
| **Power Words** | "Ultimate", "Essential", "Proven" |
| **Bucket Brigades** | "Here's the deal:", "But wait:" |
| **Examples** | "For instance, [company] did X" |

### 3.3 Introduction Formula

1. **Hook**: Surprising fact or question.
2. **Problem**: Acknowledge reader's pain.
3. **Promise**: What they'll learn.
4. **Proof**: Why you're credible.

## Part 6: Best Practices Checklist

### ✅ Do This

- ✅ **Research First**: Know what's ranking before writing.
- ✅ **Add Visuals**: Images, charts, screenshots every 300 words.
- ✅ **Update Old Posts**: Refresh for freshness signals.

### ❌ Avoid This

- ❌ **Keyword Stuffing**: Reads unnaturally.
- ❌ **Wall of Text**: Break up with visuals and formatting.
- ❌ **No CTA**: Every post needs a next step.
