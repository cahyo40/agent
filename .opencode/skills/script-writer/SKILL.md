# Script Writer

## Overview

Write compelling video scripts that hook viewers and drive engagement.

## When to Use This Skill

- Use when writing YouTube scripts
- Use when creating video content

## How It Works

### Step 1: Script Structure

```markdown
## Video Script Template

### HOOK (0-30 seconds)
[Pattern interrupt + promise]
"What if I told you that [unexpected claim]..."

### INTRO (30s-1min)
- Quick context
- Why should they care
- What they'll learn

### BODY (Main content)
Point 1: [Topic]
- Explain concept
- Give example
- Transition

Point 2: [Topic]
- Explain concept
- Give example
- Transition

Point 3: [Topic]
- Explain concept
- Give example

### CONCLUSION
- Recap main points
- Call to action
- Tease next video
```

### Step 2: Hook Formulas

```markdown
## Proven Hook Templates

### Curiosity Gap
"The truth about [topic] that nobody tells you..."

### Bold Claim
"This one change [increased/saved] [metric] by [%]"

### Story Hook
"Three years ago, I [relatable struggle]..."

### Controversy
"Unpopular opinion: [contrarian take]"

### Direct Address
"If you're struggling with [problem], watch this"
```

### Step 3: Writing Tips

```markdown
## Engagement Techniques

### Retention Boosters
- "Here's the thing..."
- "But wait, it gets better..."
- "Stay until the end for..."

### Pattern Interrupts
- Change topic angle
- Ask rhetorical questions
- Use unexpected transitions

### Conversational Tone
- Write like you speak
- Use "you" frequently
- Short sentences
```

## Best Practices

- ✅ Hook in first 5 seconds
- ✅ One idea per sentence
- ✅ Read aloud before recording
- ❌ Don't use jargon
- ❌ Don't ramble
