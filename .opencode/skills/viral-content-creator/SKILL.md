# Viral Content Creator

## Overview

Create content optimized for virality and sharing.

## When to Use This Skill

- Use when maximizing reach
- Use when riding trends

## How It Works

### Step 1: Viral Triggers

```markdown
## Why Content Goes Viral

### Emotional Triggers
- Awe (inspiring)
- Amusement (funny)
- Anxiety (urgent)
- Anger (outrage)
- Surprise (unexpected)

### Share Motivations
- Identity: "This is so me"
- Social currency: "I found this first"
- Practical value: "This is useful"
- Stories: "You won't believe..."
```

### Step 2: Content Formats

```markdown
## High-Share Formats

| Format | Share Rate |
|--------|------------|
| Listicles | High |
| How-tos | Medium-High |
| Infographics | High |
| Quizzes | Very High |
| Memes | Very High |
| Controversies | Very High |
```

### Step 3: Trend Riding

```markdown
## Trend Spotting

### Sources
- Twitter Trending
- TikTok Discover
- Google Trends
- Reddit Rising

### Timing
- Early adopter: 2-6 hours
- Optimal: 6-24 hours
- Late: 24-48 hours
- Dead: 48+ hours
```

## Best Practices

- ✅ Move fast on trends
- ✅ Add unique angle
- ✅ Evoke strong emotions
- ❌ Don't force irrelevant trends
- ❌ Don't be offensive
