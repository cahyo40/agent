# Freelance Business Manager

## Overview

This skill helps you run a successful freelance business by creating professional quotations, managing client relationships, setting competitive rates, and handling invoicing and contracts.

## When to Use This Skill

- Use when creating project quotations/proposals
- Use when determining hourly or project rates
- Use when drafting contracts and agreements
- Use when creating invoices
- Use when managing client communication
- Use when negotiating project scope
- Use when handling payment terms

### Step 6: Project Pricing Models

```markdown
## Pricing Models Comparison

| Model | Best For | Pros | Cons |
|-------|----------|------|------|
| **Hourly** | Unclear scope, maintenance | Flexible, fair for changes | Client may worry about costs |
| **Fixed Price** | Well-defined scope | Predictable for client | Risk of under-estimating |
| **Value-Based** | High-impact projects | Higher earning potential | Harder to justify |
| **Retainer** | Ongoing work | Stable income | May limit other clients |
| **Milestone** | Large projects | Spreads risk | Requires clear milestones |

### When to Use Each

**Hourly Rate:**
- Ongoing maintenance
- Consulting/advisory
- Unclear requirements
- Rate: $[X]/hour + time tracking

**Fixed Price:**
- Clear deliverables
- Defined timeline
- Client wants predictability
- Add 20-30% buffer

**Value-Based:**
- Project generates clear ROI
- Example: "This will save you $50K/year"
- Price at 10-20% of value: $5K-10K
- Requires confident positioning

**Retainer:**
- Regular monthly hours
- Example: 20h/month @ $55 = $1,100
- Discount vs hourly (10-15%)
- Priority support included
```
