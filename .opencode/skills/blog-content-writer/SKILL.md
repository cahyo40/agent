# Blog Content Writer

## Overview

Write SEO-optimized blog posts that rank and engage readers.

## When to Use This Skill

- Use when creating blog content
- Use when optimizing for SEO

## How It Works

### Step 1: Article Structure

```markdown
## Blog Post Template

### Title (H1)
- Include primary keyword
- Under 60 characters
- Create curiosity

### Introduction
- Hook reader in first line
- State the problem
- Promise the solution

### Body (H2 sections)
- H2: Main topic 1
  - H3: Subtopic
  - Content + examples
- H2: Main topic 2
  - H3: Subtopic
- H2: Main topic 3

### Conclusion
- Recap key points
- Call to action
- Related content links
```

### Step 2: SEO Optimization

```markdown
## On-Page SEO

### Keyword Placement
- Title (H1)
- First 100 words
- At least 1 H2
- Meta description
- URL slug

### Technical
- Meta description: 150-160 chars
- URL: short, keyword-rich
- Images: alt text with keywords
- Internal links: 3-5 per post
```

### Step 3: Readability

```markdown
## Writing Guidelines

- Sentences: 20 words max
- Paragraphs: 3-4 lines max
- Use bullet points
- Add images every 300 words
- Flesch score: 60+
```

## Best Practices

- ✅ Research keywords first
- ✅ Answer search intent
- ✅ Use data and examples
- ❌ Don't keyword stuff
- ❌ Don't skip editing
