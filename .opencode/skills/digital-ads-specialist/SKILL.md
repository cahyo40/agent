# Digital Ads Specialist

## Overview

Create and optimize digital advertising campaigns for maximum ROI.

## When to Use This Skill

- Use when setting up ad campaigns
- Use when optimizing ad performance

## How It Works

### Step 1: Campaign Structure

```markdown
## Google Ads Structure

Account
└── Campaign (Budget, Targeting)
    └── Ad Group (Keywords, Audience)
        └── Ads (Creative, Copy)

## Campaign Types
- Search: Text ads on Google Search
- Display: Banner ads across websites
- Shopping: Product listing ads
- Video: YouTube ads
- Performance Max: AI-optimized across all

## Meta Ads Structure
Campaign (Objective)
└── Ad Set (Audience, Budget, Placement)
    └── Ad (Creative, Copy, CTA)
```

### Step 2: Audience Targeting

```markdown
## Google Ads Targeting

### Keywords
- Broad match: running shoes
- Phrase match: "running shoes"
- Exact match: [running shoes]
- Negative: -free, -cheap

### Audiences
- In-market: actively researching
- Affinity: interests/lifestyle
- Custom: your own lists
- Remarketing: past visitors

## Meta Ads Targeting

### Demographics
- Age, gender, location
- Language, education
- Job title, income

### Interests & Behaviors
- Hobbies, activities
- Purchase behavior
- Device usage

### Custom Audiences
- Website visitors (Pixel)
- Customer list upload
- App activity
- Engagement
```

### Step 3: Key Metrics

```markdown
## Performance Metrics

| Metric | Formula | Good Benchmark |
|--------|---------|----------------|
| CTR | Clicks / Impressions | 2-5% (Search) |
| CPC | Cost / Clicks | Varies by industry |
| CVR | Conversions / Clicks | 2-5% |
| CPA | Cost / Conversions | Depends on LTV |
| ROAS | Revenue / Ad Spend | 3-5x minimum |

## Quality Score (Google)
- Expected CTR
- Ad relevance
- Landing page experience
- Score: 1-10 (aim for 7+)
```

### Step 4: Optimization Tips

```markdown
## Quick Wins

### Copy
- Include keyword in headline
- Clear value proposition
- Strong CTA
- Use numbers (50% off, 24-hour)

### Targeting
- Start narrow, expand later
- Use negative keywords
- Exclude poor performers
- Lookalike audiences

### Budget
- Start low, scale winners
- Dayparting (best hours)
- Geo-optimization
- Device bid adjustments
```

## Best Practices

- ✅ A/B test ad creatives
- ✅ Use conversion tracking
- ✅ Optimize for ROAS, not clicks
- ❌ Don't ignore negative keywords
- ❌ Don't set and forget
