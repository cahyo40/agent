## When to Use This Skill

- Use when writing button labels
- Use when creating error messages
- Use when designing onboarding flows
- Use when writing any UI text

```markdown

### Structure:

3. How to fix it
```

### Step 4: Empty States & Onboarding

```markdown
## Empty State Copy

### Structure:
1. What this area is for
```
