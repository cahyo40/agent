# UI/UX Generator Workflow

## Overview

Design-to-code pipeline that takes UI/UX reference images and converts them to production-ready source code. Covers reference input, design evaluation, design system generation, Stitch AI prompting, screen generation, code conversion, and QA.

## When to Use

- Use when the user provides UI/UX design references and wants code
- Use when converting designs to production-ready components
- Use when using Stitch AI for UI generation
