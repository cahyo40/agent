## Overview

Creates structured learning materials from user requests. Follows a 5-phase pipeline: brainstorming, roadmap creation, material development, output formatting, and review.

## When to Use

- Use when asked to create learning materials or educational content
- Use when the user wants a structured curriculum or tutorial series
